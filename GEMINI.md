# CLAUDE.md — MCP Slack

## Projeto

MCP server para integracao com Slack — listar canais, ler e postar mensagens. Construido com FastMCP.

## Stack

Python | FastMCP | Slack SDK

## Status

Ativo. Ferramenta interna.

## Setup obrigatório após clone

Executar uma vez: `git config core.hooksPath .githooks`

Isso ativa o pre-push hook que exige bump do arquivo `VERSION` quando há mudanças de código.

Portugues Brasileiro. Termos tecnicos em ingles.
